# Bluetooth-device-monitoring
***请确保设备拥有心率广播功能***
**请确保运行本软件的设备上拥有蓝牙功能**
搭配[Smart_Wearable_Warning_Platform](https://github.com/2er0oZ/Smart_Wearable_Warning_Platform)使用

## 使用说明
* 打开可穿戴设备的心率广播功能。
* 打开本软件，点击扫描设备，扫描完成后会将设备显示至设备列表（使用前需打开蓝牙功能）。
* 点击选择设备列表中的设备后，点击连接设备。
* 连接设备后会在状态栏处更新设备详情。并且会在心率数据处看到实时心率数据。
* 可以点击“开始记录”进行记录心率。
* 需要导出心率时，请先停止记录。随后进行导出。

心率导出的CSV文件格式如下

| 时间戳 | 日期时间 | 心率值(bpm) | 步频 |
| :---: | :---: | :---: |  :---: |
| 1772323200.0 | 2026-03-01 00:00:00 | 51bpm | 0 |
